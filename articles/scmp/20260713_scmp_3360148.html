Beyond Claude Code: the Chinese AI tools poised to benefit after back-door alert

SCMP | FREE DEEP | 653 words | 2026-07-13

Beijing’s recent cybersecurity warning against American artificial intelligence lab Anthropic is expected to accelerate a shift among Chinese developers towards domestic coding alternatives, according to analysts.

China’s National Vulnerability Database (NVDB), overseen by the Ministry of Industry and Information Technology, issued an alert this week claiming multiple versions of Anthropic’s flagship Claude Code tool contained a security “back door”.

According to the agency, the software could send user locations and identities to remote servers without consent. The NVDB urged local organisations to uninstall the affected versions immediately or upgrade to patched releases.

The warning followed Anthropic’s acknowledgement last week that it had embedded a tracking code into Claude Code to prevent the illicit “distillation” – or unauthorised copying – of its models.

Responding to the NVDB’s alert on Wednesday, Anthropic said its usage policy had always barred users based in China from accessing its services.

Cai Peng, a Beijing-based cybersecurity partner at Zhong Lun Law Firm, said he expected more Chinese companies to abandon foreign AI tools, driven by mounting security concerns and the country’s “strategic imperative” for tech self-reliance.

Here are some of the most prominent Chinese AI coding assistants filling the void.

Trae (ByteDance)

Developed by ByteDance, Trae is an AI-native integrated development environment (IDE) designed to help software engineers build software using natural-language prompts and autonomous agents.

Like Claude Code, it can generate code, debug software, explain complex codebases and build applications from scratch. Its “Builder” mode enables end-to-end project creation, while a built-in chat interface supports iterative problem-solving.

Trae is currently one of China’s most widely adopted AI coding assistants, with more than 6 million registered users and 1.6 million monthly active users by the end of 2025, according to ByteDance.

The tool’s main advantage is localisation: the China-facing version supports popular local foundation models such as Doubao and DeepSeek and allows users to connect their own model application programming interfaces (APIs).

Qoder CN (Alibaba)

Qoder is Alibaba Group Holding’s agentic coding platform, which evolved from the company’s Tongyi Lingma assistant before launching globally in 2025.

Positioned as a comprehensive AI development workspace rather than a simple coding assistant, it supports AI-powered code generation, project analysis and autonomous task execution.

Qoder also allows international users to connect their preferred foundation models through custom API keys and integrates with third-party enterprise development tools. Alibaba owns the South China Morning Post.

CodeBuddy (Tencent)

CodeBuddy is Tencent Cloud’s flagship AI coding assistant, powered by the company’s Hunyuan large language model alongside Chinese AI start-up DeepSeek’s models.

The platform supports more than 200 programming languages – including Python, Java, Go and JavaScript – offering contextual code completion, debugging and refactoring.

It is also designed to help users break down software requirements, generate code across multiple files and handle testing.

CodeGeeX and ZCode (Zhipu AI)

Developed by the high-flying lab Zhipu AI, CodeGeeX was one of China’s earliest AI coding assistants and boasts a sizeable local developer community thanks to its cross-platform compatibility with major IDEs.

CodeGeeX is positioned primarily as a foundational code model and developer assistant. Highlighting China’s push for hardware self-reliance, Zhipu said the model was trained using Huawei Technologies’ MindSpore framework on Ascend AI processors, offering developers a home-grown alternative.

Also from the Zhipu stable, ZCode is an agentic coding workspace designed to automate the software development life cycle, from initial requirement planning to debugging, testing and documentation.

Built as the official development environment for Zhipu’s advanced GLM-5.2 model, ZCode boasts a context window of up to 1 million tokens, allowing it to analyse large codebases more efficiently.

Unlike conventional AI coding assistants, ZCode is said to be capable of orchestrating entire workflows. It features a multi-model interface, allowing developers to switch between Chinese and international models – including those from OpenAI and Google – within a single project.
